# Cartes de visite - Le Patis

Trois pistes graphiques au format SVG horizontal `1050 x 600`, ratio carte de visite classique.

- `le-patis-carte-01-nature.svg` : direction nature contemporaine, claire, avec motif topographique.
- `le-patis-carte-02-photo.svg` : direction photo immersive, basée sur la photo principale du site.
- `le-patis-carte-03-premium.svg` : direction premium sobre, fond sombre et monogramme.
