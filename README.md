# Hotel House Alpes Mancelles

Site vitrine single page pour une location courte duree dans les Alpes
Mancelles.

## Prendre en main

Installer les dependances :

```bash
npm i
```

Lancer le serveur de developpement :

```bash
npm run dev
```

Verifier la compilation :

```bash
npm run build
```

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn/ui

## Structure

```text
src/
  main.tsx
  app/
    App.tsx
    components/
    pages/
  assets/
  styles/
```
